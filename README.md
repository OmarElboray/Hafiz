# Hafiz (حافظ)

### Overview
Hafiz is an intelligent Quran memorization assistant designed to help users master their Hifz through structured repetition and active testing. Unlike standard reading apps, Hafiz focuses on the cognitive process of memorization, providing a digital environment for both initial learning and long-term retention.

### Core Features
* **Repetition & Connection:** Automated cycles for "Tikrar" (repetition) to solidify new verses before moving forward.
* **Tasmi3 Mode (Recitation Test):** A specialized interface to test memorization, allowing users to verify their accuracy and identify "Mutashabihat" (similar verses).
* **Progress Tracking:** Data-driven insights into which Surahs are well-retained and which require "Muraja'ah" (revision).

### Tech Stack
* **Frontend:** JavaScript 
* **Logic:** Custom algorithms for spaced repetition and memorization intervals.
* **Localization:** Fully optimized for Arabic script and RTL (Right-to-Left) layouts.

> [!IMPORTANT]
> **Status: Under Active Development**
> We are currently refining the Tasmi3 logic and building the internal database for verse connection. The codebase remains private during the initial prototyping phase.

### Mission
The goal of Hafiz is to utilize modern software engineering to serve the ancient tradition of Quranic Hifz, making the process more systematic for students of all levels.
